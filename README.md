# Java High Suit Card Game

A simple Java card game where players draw cards, and the winner is determined by the **highest suit and value**. This project demonstrates basic Java programming concepts including OOP, arrays, loops, and game logic.

---

## 📝 Features

- Play a classic “High Suit” card game
- Compare card values and suits
- Command-line interface
- Object-Oriented Java design

---

## 🛠️ Technologies Used

- **Java 8+**
- No external libraries required
- IDE friendly (NetBeans, IntelliJ IDEA, Eclipse, etc.)
